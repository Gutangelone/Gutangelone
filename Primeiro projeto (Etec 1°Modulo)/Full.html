Full-stack

O que faz um desenvolvedor Full-stack?

Como sua própria denominação indica, o desenvolvedor Full-stack é o profissional habilitado para compreender e operar em todas as camadas do desenvolvimento de um projeto, desde a criação de servidores internos até interfaces de comunicação com o usuário final. Sendo bom em fazer tanto a parte do visual e da interação do usuário em um site ou aplicativo (o front-end) quanto a parte de armazenamento e processamento de informações no servidor (o back-end).

Principais Linguagens

PHP: È uma linguagem de script open source de uso geral, muito utilizada, e especialmente adequada para o desenvolvimento web e que pode ser embutida dentro do HTML.
JavaScript: È uma linguagem de programação usada popularmente que permite aos usuários implementar recursos sofisticados em páginas da web. Ele pode ser acionado sempre que uma página estiver realizando uma ação dinâmica, como mostrar atualizações periódicas de conteúdo, gráficos animados bidimensionais e tridimensionais, mapas interativos, caixas de vídeo e muito mais.
HTML/CSS: HTML é a linguagem de marcação padrão usada para criar a base de uma página web. Já o CSS é usado para criar o layout da página, cor, fontes e estilo. Ambas as linguagens são absolutamente essenciais para se tornar um desenvolvedor front-end.
